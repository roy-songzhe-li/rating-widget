How to Integrate the Rating Widget

This example shows how to integrate the Rating Widget into your website. Simply add the script tag and the custom element to your HTML.

Step 1: Add the Script

<script src="https://your-domain.vercel.app/rating-widget.umd.js"></script>

Step 2: Add the Widget

<rating-widget max-rating="5" theme="light" callback-url="https://your-api.com/ratings"></rating-widget>

Available Attributes

  • max-rating: The maximum number of stars (default: 5)
  • theme: The theme of the widget (default: "light")
  • callback-url: The URL to send the rating data to (optional)

Event Listening

You can listen for the rating-submitted event to get the rating value: 

document.addEventListener('rating-submitted', function(e) {
  console.log('Rating submitted:', e.detail.rating);
  // Do something with the rating
});
